单引号和双引号区别

  1. 单引号不解析变量
  1. 双引号解析变量时, 应该加大括号 {变量名}

  1. 如果没有变量需要解析, 尽量使用单引号, 因为单引号的执行效率高

  2. 单引号只解析 ‘ 和 ** , 双引号还可以解析\r\n\t转义字符.

变量

特性

1.变量是存放数据的容器.

2.它里面的数据可以多次读取使用.

3.把新的数据放进去, 旧的数据会被覆盖. 也就是说它存的内容是可以变化的.

命名规则

1.以$开头

2.数字、字母、下划线的组合，但不能以数字开头

3.严格区分大小写

4.可以使用中文，但不建议

命名方法

1.驼峰命名法：

​ 单驼峰：第一个单词首字母小写，以后每个单词的首字母都大写。

​ 双驼峰：每个单词的首字母都大写。

2.使用下划线。每个单词之间使用下划线来进行分隔。

类型

变量中存放的数据类型, 就是变量的类型. php中变量有8种类型:

4种标量 (标准的变量类型)

  • 布尔型 (boolean)
  • 整数型 (integer)
  • 浮点型 (float)
  • 字符串型 (string)

2种复合类型 ( 复合：里面可以放多个值)

  • 数组 ( array )
  • 对象 ( object )

2种特殊类型

  • 资源 ( resource )
  • NULL

打印变量的类型和值可以用 var_dump( 变量名 );

布尔类型 ( boolean )

表明两个状态. 用来做开关. 用来做判断条件.

true 真 你们比哥年轻

false 假 哥喜欢男人.

is_bool( 变量名 ) 可以判断变量是否为布尔类型 是,返回true 不是,返回false

​

整数类型 ( integer )

十进制 : 0 ~ 9 默认

二进制 : 0 ~ 1 以0b开头

十六进制 : 0 ~ f 以0x开头

八进制 : 0 ~ 7 以0开头

当用 echo 输出时, 它只会以十进制数显示.

is_int( 变量名 )可以判断变量是否为整数类型

​

浮点类型 ( float )

浮点数, 即小数. 受存储精度影响, 两个小数的运算结果很多时候也是不准确的.如 0.1 + 0.2 就不等于 0.3

is_float( 变量名 )可以判断变量是否为 浮点型

is_numeric( 变量名 )可以判断变量是否为 数值类型

NULL类型

表示一个变量没有值。NULL类型唯一可能的值就是NULL

被认为是NULL的情况:

is_null()，用来判断变量是否为空，如果是返回true，否则返回false。

unset( 变量名 )

销毁指定的变量

1.变量的实质是内存空间的一块存储区域.

2.PHP代码结束时,所有变量常量都会自动销毁

3.变量不再使用时, 手动提前销毁, 可以及时释放宝贵的内存空间

isset( 变量名 )

检测变量是否已设置并且非 NULL

​ 1) 变量未声明 或者 未赋值

​ 2) 变量的值为 null

上面两种情况返回 false , 其他情况返回 true

​

empty( 变量名 )

检查一个变量是否为空

为空返回 true 不为空返回 false

为空的情况有以下几种 :

  • “” (空字符串)
  • 0 (整数 0)
  • 0.0 (浮点数 0)
  • “0” (字符串 0)
  • NULL
  • FALSE (布尔型)
  • array() (空数组)
  • $var; (声明未赋值的变量)

递增/递减运算符

对于布尔型变量, 自增自减都是原来的值

单个字符时, 会变成下一个字符

赋值运算符

比较运算符

最后结果为布尔型

逻辑运算符

最后结果为布尔型

&&(and) 逻辑与 符号两边同时为 true 结果就为true, 否则为 false

||(or) 逻辑或 符号两边只要一边为 true 结果就为 true. 仅当两边同时为 false 时, 结果才为 false

! 逻辑非 将值取反, 原来的 true 变成 false ,原来的 false 变成 true

xor 逻辑异或 符号两边不相同的时候为 true 两边相同的时候为false $a = (true xor true);

短路

或短路 当 || 左边的表达式为真时, 结果肯定为真, 所以 || 右边的表达式就不执行了

与短路 当 && 左边的表达式为假时, 结果肯定为假, 所以 && 后边的表达式就不执行了

三元运算符

$a = 表达式1 ? 表达式2 : 表达式3；

表达式1为真时，把表达式2的值赋值给$a;

表达式1为假时，把表达式3的值赋值给$a;

其他运算符

@ 错误抑制符 在发生错误的代码前加上@, 页面上就不会显示错误信息了

`` 执行命令符 ` 系统自带命令 `; 通过这种形式可以执行系统自带的命令
